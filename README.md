# Advancing Circular Economy Practices in Agricultural Waste Management Across Selected African Countries

## Data Analysis Capstone Project

### Project Overview

This project applies data analytics and visualization techniques to evaluate agricultural waste generation, post-harvest food loss, and socioeconomic indicators across selected African countries. The objective is to identify circular economy opportunities that can promote sustainable development, green job creation, environmental sustainability, and evidence-based policymaking.

The project culminates in an interactive Power BI dashboard that transforms multiple datasets into actionable insights for policymakers, investors, researchers, and development organizations. :contentReference[oaicite:0]{index=0}

---

## Project Objectives

The project aims to:

- Quantify agricultural waste and post-harvest food loss by country and crop.
- Integrate socioeconomic indicators such as unemployment, poverty, and greenhouse gas emissions.
- Identify regions with high potential for circular economy investments.
- Present findings through an interactive Power BI dashboard for decision support. :contentReference[oaicite:1]{index=1}

## Key Findings

The analysis revealed that:

- Agricultural waste varies significantly across countries.
- Countries with high waste volumes often experience higher unemployment and poverty.
- Storage and post-harvest activities account for the largest food losses.
- Converting agricultural waste into compost, animal feed, or biogas presents significant economic and environmental opportunities.
- Central and West African countries show strong potential for circular economy investments. :contentReference[oaicite:7]{index=7}
